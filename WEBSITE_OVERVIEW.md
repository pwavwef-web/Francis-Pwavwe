# Francis Pwavwe Personal Website

## Overview
A modern, professional personal website showcasing Francis Pwavwe as a Strategic Thinker, Tourism Professional, and Digital Innovator.

## Structure
- **index.html** - Main HTML structure (21KB)
- **styles.css** - Complete styling with blue theme (19KB)
- **script.js** - Interactive JavaScript features (7.3KB)

## Sections
1. **Hero**: Bold headline with professional positioning
2. **About**: Academic excellence, innovation, and leadership highlights
3. **Projects**: AZ Learner (featured) + tourism and leadership projects
4. **Skills**: Tourism, research, and leadership capabilities with progress bars
5. **Experience**: Timeline of key positions and achievements
6. **Vision**: Bold statement about international tourism strategy
7. **Contact**: Email, LinkedIn, and functional contact form

- Deep royal blue (#0F3D7A, #0A2647) with gold accents (#D4AF37)
- Typography: Playfair Display (headings) + Inter (body)
- Smooth scrolling navigation
- Scroll-triggered animations
- Mobile responsive
- Professional, founder-level aesthetic
