## Quick Actions

### Track an Expense
1. Go to **Finances** tab
2. Click **+ Add Transaction**
3. Enter amount, category (e.g., "Food"), and description
4. Select type: "Expense"
5. Click **Add Transaction**

### Create a Task
1. Go to **Planning** tab
2. Click **+ Add Task**
3. Enter title, description, priority, and due date
4. Click **Add Task**

### Write a Blog Post
1. Go to **Blogs** tab
2. Click **+ Write New Blog**
3. Enter title and use the editor toolbar for formatting
4. Click **Publish Blog**

### Upload a Project
1. Go to **Projects** tab
2. Enter project name and description
3. Click the upload area to select files
4. Click **Upload Project**

### Get AI Advice
- In **Finances** tab, use the AI Financial Advisor chat
- In **Planning** tab, use the AI Planning Assistant chat
- Ask questions like:
  - "How can I save money on food?"
  - "Help me prioritize my tasks"
  - "Create a budget for a student"
